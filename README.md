# EasyRSGain 2026 🚀
Herramienta profesional para la normalización de audio (LUFS) y edición masiva de metadatos en Debian.

## Características
- **Normalización Inteligente**: Basada en rsgain para un volumen perfecto.
- **Editor de Tags**: Edición masiva de Artista, Álbum y Año para cientos de canciones.
- **Soporte de Carátulas**: Aplica portadas .jpg/.png a álbumes completos con un clic.
- **Interfaz Moderna**: Optimizada para XFCE y entornos oscuros.

## Instalación en Debian
Descarga el archivo `.deb` y ejecuta:
\`\`\`bash
sudo apt install ./easyrsgain-suite_1.1_all.deb
\`\`\`
